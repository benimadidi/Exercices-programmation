BUSY Bar Boutique Téléchargements Blog

BUSY Bar est un dispositif multi-outil de productivité avec un écran LED pixelisé. Minuteur de concentration avec fonction ce blocage des distractions sur votre téléphone et PC. Entièrement personnalisable, open-source et prêt pour la maison connectée.

Outil de productivité

Blocage des distractions, message BUSYpersonnalisé, minuteur de concentration, synchronisation multiplateforme

Applications et intégration

Bibliothèque d’applications, connexion à des logiciels tiers, intégrations avec les événements de calendrier et les appels

Support maison connectée

Connectez-vous à Google Home et Apple Home via le protocole Matter

Pour les développeurs

API HTTP ouverte, SDK open-source, bibliothèque Python/Go/JavaScript sans verrouillage fournisseur

Montrez que vous êtes occupé

Empêchez les autres d'interrompre votre concentration profonde. Lécran affiche que vous êtes occupé et quand vous serez à nouveau disponible.

Activation instantanée

Activez et desactivez le statut BUSY manuellement avec un large bouton. Fonctionne entièrement hors ligne, sans logiciel supplémentaire requis.

Set timer

Set an optional expiration BUSY using the scroll wheel to let others know when you'll be available again.
